# Contact Form Project

A professional contact form demonstrating front-end development skills with modern web technologies.

##  Project Overview
This project showcases proficiency in HTML, CSS, and JavaScript through a fully-functional contact form with validation and responsive design.

## Features
- **Modern UI Design**: Gradient background with smooth animations
- **Form Validation**: Real-time input validation with error messages
- **Responsive Layout**: Mobile-first design that works on all devices
- **User Feedback**: Loading states, success/error notifications
- **Clean Code Structure**: Well-organized, commented codebase

## Technologies Used
- **HTML5**: Semantic markup and form structure
- **CSS3**: Flexbox, Grid, CSS Variables, responsive design
- **JavaScript (ES6+)**: Form handling, validation, DOM manipulation
- **Font Awesome**: Icon library for visual elements

## How to Run
1. Open `index.html` in any modern web browser
2. Fill out the contact form
3. Experience real-time validation
4. Submit to see success feedback
